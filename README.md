# devtube

Play Youtube music via command line

<p align="center">
	<br>
	<img src="assets/devtube.svg" width="800">
	<br>
</p>

## Install

```bash
npm install -g devtube
```

## Usage

```bash
devtube <playlist>
```

Example: `devtube "https://www.youtube.com/watch?v=xz7TbHXXV3o&list=RDxz7TbHXXV3o&start_radio=1"`
