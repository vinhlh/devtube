## Features

- No ads.

- No auto paused (experiment).

## Install

```bash
npm install -g devtube
```

## Usage

```bash
devtube <playlistOrSearchKeyword>
```

Example: `devtube "https://www.youtube.com/watch?v=VjHMDlAPMUw"`

`devtube "Faded"`
